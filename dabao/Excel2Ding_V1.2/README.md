# Excel2Ding

Excel2Ding 是一个用于处理 Excel 文件的应用程序，提供了用户友好的图形界面，允许用户轻松导入、处理和导出 Excel 数据。

## 项目结构

- `src/Excel2Ding_V1.2.py`: 主应用程序代码，包含 GUI 和 Excel 处理逻辑。
- `src/Excel2Ding.ico`: 应用程序的图标文件。
- `src/requirements.txt`: 项目所需的依赖项列表，可以使用 pip 安装。
- `setup.py`: 用于打包应用程序的设置脚本。
- `build.spec`: 用于定义如何构建可执行文件的配置文件。

## 安装依赖

在项目根目录下运行以下命令以安装所需的依赖项：

```
pip install -r src/requirements.txt
```

## 打包应用程序

使用以下命令打包应用程序：

```
pyinstaller --onefile --icon=src/Excel2Ding.ico setup.py
```

## 运行应用程序

打包完成后，您可以在 `dist` 文件夹中找到生成的可执行文件，双击运行即可启动应用程序。

# 欢迎使用 Excel 数据处理小工具！
你好！感谢你使用这款 Excel 数据处理小工具。这个小工具可以帮助你快速处理 Excel 文件，并自动保存结果。以下是使用说明和一些注意事项，希望能帮助你顺利使用它！
________________________________________
## 工具功能
1.	自动读取路径：
从【file_path.txt】文件中读取输入文件和输出文件的路径。
2.	数据处理：
根据预设规则处理 Excel 文件（例如删除特定条件的行）。
3.	自动保存：
将处理后的数据保存到指定路径。

________________________________________
## 使用步骤
1. 准备飞书数据
向有飞书导出数据权限的人员【黄漫】要一份，包含【产品设计部】与【智慧会议BG】部门的特做数据Excel表格。

2. 手动处理
    1. 手动将创建时间格式调整为文本，如 45757.6148148148
    2. 请将无用表去除，仅保留一份需要进行清洗的表格
    3. 请删除第一行的说明信息
 

3. 准备文件
确保你的 【输入文件.xls】
 
4. 运行工具
双击 【特做数据自动整理.exe】 文件运行工具。
请按操作指引选择对应的文件，筛选时间，输出文件地址等信息，点击开始处理即可等待结果
5.  查看结果
打开所选输出路径，查看处理后的 Excel 文件。
